Technical Overview

Production-grade AI platform built for reliability, scalability, and developer velocity.

System Architecture

Web Application

Ruby on Rails 7.2 + Hotwire

Server-rendered views with Turbo Streams for real-time updates. Stimulus controllers for interactive UI. No SPA complexity — fast page loads, great SEO.

AI Orchestration Layer

Multi-Provider LLM Integration

Unified interface to OpenAI GPT-4o, Anthropic Claude 3.5 Sonnet, and Google Gemini. Automatic model selection per task. Streaming responses via Turbo.

Background Processing

Sidekiq + Redis

AI generation, document parsing, email campaigns, and PPTX exports run asynchronously. Job retries, dead letter queue, and cron scheduling built in.

Document Engine

Rubyzip XML Generation

DOCX and PPTX generation via direct Open XML manipulation. No external dependencies. Professional formatting preserved across all exports.

AI Model Selection Strategy

Different tasks route to different AI providers for optimal quality and cost:

  • GPT-4o: Resume rewriting, pitch deck generation, complex content (highest quality)
  • Claude 3.5 Sonnet: Cover letters, interview prep, long-form analysis (best for nuance)
  • Gemini: Keyword extraction, ATS scoring, summarization (cost-efficient at scale)

Average AI cost per optimization: $0.02 — $0.08. At current pricing, gross margin is 85%+.

Scalability & Performance

Current Architecture Supports:

  • 10,000+ concurrent users — Puma multi-threaded server, database connection pooling
  • 100,000+ AI requests/month — Sidekiq background jobs prevent request blocking
  • Sub-second page loads — Server-rendered HTML, no client-side framework overhead
  • Horizontal scaling — Stateless application servers, can add workers on demand
  • 99.9% uptime target — Health checks, auto-restart, zero-downtime deploys

Infrastructure cost at current scale: ~$20/month (Hetzner Cloud). Scales to 50K users at ~$150/month.
